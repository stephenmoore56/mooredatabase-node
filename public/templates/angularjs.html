What is AngularJS?

The front end of this site was developed using AngularJS. AngularJS is an open-source JavaScript framework, maintained by Google, that assists with running single-page applications (SPA's). Its goal is to augment browser-based applications with model–view–controller (MVC) capability, in an effort to make both development and testing easier.
