Welcome / Birding Demos

Welcome to Steve Moore's portfolio Node.js site, running on Heroku. The code for this site is availabe for review in a public repo on GitHub.

The birding demos on this site demonstrate:

  • Integration of a Node.js app running on Heroku with an Amazon RDS MySQL backend; the node module mysql is used for data access.
  • The tables are drawn client-side using an AngularJS MVC app, with JSON data obtained from the Node.js app.
  • The charts are drawn client-side using Plotly.js with JSON data obtained from the Node.js app.

What is Node.js?

Node.js is a software platform that is used to build fast, scalable network (especially server-side) applications. Node.js is built on the Chrome V8 JavaScript runtime. Node.js utilizes JavaScript as its scripting language, and achieves high throughput via non-blocking I/O and a single-threaded event loop.

Node.js contains a built-in HTTP server library, making it possible to run a web server without the use of external software, such as Apache, and allowing more control of how the web server works. Node.js enables web developers to create an entire web application in JavaScript, both server-side and client-side.
