What is Node.js?

Welcome to Steve Moore's portfolio Node.js site, running on Heroku.

Node.js is a software platform that is used to build fast, scalable network (especially server-side) applications. Node.js is built on the Chrome V8 JavaScript runtime. Node.js utilizes JavaScript as its scripting language, and achieves high throughput via non-blocking I/O and a single-threaded event loop.

Node.js contains a built-in HTTP server library, making it possible to run a web server without the use of external software, such as Apache or Lighttpd, and allowing more control of how the web server works. Node.js enables web developers to create an entire web application in JavaScript, both server-side and client-side.

Node.js is cool.

Node Development

I do my development in Aptana Studio 3 on an Ubuntu laptop and on a Mac running OS X Lion. I use git and GitHub for version control. This site runs on Heroku.

The front end makes use of AngularJS, Bootstrap, jQuery and jQuery UI; most of the JavaScript was authored using CoffeeScript. The CSS was authored and pre-processed using SASS and Compass.

The back end uses redis for a session store, a MongoDB collection for user authentication, and does ordinary database queries against a remote MySQL database (in AWS RDS) on the server side and client-side via AJAX. JavaScript was authored using CoffeeScript.

Birding Demos

The birding demos demonstrate:

  • Integration of a Node.js app running on Heroku with an Amazon RDS MySQL backend; the node module mysql was used for data access.
  • The tables are drawn client-side using an AngularJS MVC app, with JSON data obtained via AJAX calls to the Node.js app.
  • The charts (not visible on phones) are drawn client-side using the Google Charts API with JSON data obtained via AJAX calls to the Node.js app.
